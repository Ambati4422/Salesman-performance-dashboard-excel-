Salesman Performance Dashboard (Excel)
Project Overview

This project analyzes salesperson performance using Excel. The dashboard provides insights into sales trends, target achievement, revenue generation, customer acquisition, and regional performance.

Objectives
Monitor salesperson performance.
Compare actual sales against targets.
Identify top-performing salespersons.
Analyze sales by region and product category.
Track monthly sales trends.
Support data-driven business decisions.
Tools Used
Microsoft Excel

Pivot Charts
Slicers
